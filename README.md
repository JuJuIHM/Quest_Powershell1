# Quest_Powershell1

## La liste des commandes unix et leur équivalent

-cp => Copy-Item

-rm => Remove-Item

-cd => Set-Location

-mkdir => New-Item

-man => Get-Help

-history => Get-History

-alias => Get-Alias

-cat => Get-Content 

## La commande que j'ai utilisé pour trouver les commandes est la suivante :

PS C:\Users\juju> man history
